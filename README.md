Data manipulation at scale. Working with real CSV datasets, cleaning messy data, reshaping tables, merging, grouping, and time-series analysis.

## 🚀 Getting Started

Clone the repo and set up a local environment:

```bash
# Clone
git clone https://github.com/<your-username>/python-journey.git
cd python-journey

# Create a virtual environment
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# Install dependencies
pip install jupyter flask pandas numpy matplotlib

# Launch Jupyter
jupyter notebook
```

Then open any `.ipynb` file from the folder that matches the topic you want to explore.
